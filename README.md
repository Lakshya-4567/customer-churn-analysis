# Customer Churn Analysis Dashboard

## Project Overview
This project focuses on analyzing customer churn patterns to identify why users stop using a service. Using Python and statistical visualization in Google Colab, I explored a dataset of over 7,000 customers to uncover the primary drivers of attrition.

## Key Insights Found
* **Contract Impact:** Month-to-month subscribers show the highest churn rates compared to long-term contracts.
* **Price Sensitivity:** Higher monthly charges strongly correlate with increased customer turnover.
* **Retention Strategy:** Data suggests that incentivizing long-term contracts could significantly reduce overall churn.

## Tech Stack
* **Language:** Python
* **Environment:** Google Colab
* **Libraries:** Pandas (Data Cleaning), Seaborn & Matplotlib (Visualization)
